PiotrFlix

PiotrFlix to w pełni autorska aplikacja desktopowa służąca do katalogowania filmów i seriali. Wykorzystuje PyQt6, SQLite oraz integrację z TMDB API. Aplikacja działa offline i pozwala na pełną personalizację kolekcji multimediów użytkownika. Interfejs użytkownika został zaprojektowany z myślą o prostocie, funkcjonalności i estetyce.

Przykładowy fragment kodu odpowiedzialny za pobieranie danych z TMDB API: 

import requests

def fetch_tmdb_movie(movie_id):
    url = f"https://api.themoviedb.org/3/movie/{movie_id}"
    params = {"api_key": "YOUR_API_KEY", "language": "pl-PL"}
    response = requests.get(url, params=params)
    return response.json()
