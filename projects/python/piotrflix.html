PiotrFlix to rozbudowana aplikacja desktopowa pozwalająca na kompleksowe zarządzanie kolekcją filmów i seriali. Wspiera integrację z TMDB API, umożliwia automatyczne pobieranie metadanych i grafik, a także synchronizację z Plex Media Server. Interfejs użytkownika został stworzony z wykorzystaniem PyQt6, oferując intuicyjną i nowoczesną obsługę.

Wybrane fragmenty kodu

# Pobieranie danych z TMDB
response = requests.get(f"https://api.themoviedb.org/3/search/movie?api_key=API_KEY&query={title}")
data = response.json()
title = data['results'][0]['title']

Fragment odpowiedzialny za integrację z API TMDB i pobieranie tytułu filmu.
